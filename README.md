<h3># Task Manager CLI Application </h3>

<p>
    A simple and efficient command-line based Task Manager application built with Python.
    This project allows users to organize tasks, manage deadlines, track completion status, and store data persistently using JSON.
</p>

<h2># Features </h2>

<p>
    * Add new tasks with deadlines and priority levels
    * View tasks sorted by priority and deadline
    * Mark tasks as completed
    * Edit existing tasks
    * Deadline tracking dashboard
    * Weekly productivity summary
    * Persistent task storage using JSON
    * Input validation and error handling
</p>
